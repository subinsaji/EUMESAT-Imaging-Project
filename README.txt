There are >>5 items<< that are needed to run our project code:

- metsat_img.py (custom package)
- example_uses.py (user run file)

- earth_landmask.gif (mask)
- thumbnail_africa.jpg (mask(

- images (**folder** that contained all the Infrared and Visible image data from Meteosat-9)


>>> example_uses.py imports the package metsat_img.py. So there is no need to run metsat_img.py.

>>> The user will use example_uses.py to run the individual cell blocks of code to obtain images and change slicing parameters (if they so wish). 

>>> Each cell block is labelled with a letter. Please read the commented documentation in each cell block of example_uses.py.

>>> Since the code is dealing with numerous images, which all have to be processed, please anticipate ~1-3 mins of waiting before the figures are shown (depending on what device has been used to run the code).
